Program-5: Create RDS Server and connect using MySQL Workbench

1. Install MySQL Workbench (Client) on your PC
   - Open browser → Google "MySQL Workbench download" → Download for your OS → Install.

2. Create Database in AWS RDS
   - Login AWS Console → Search "RDS" → Open RDS service.
   - Left menu → Databases → Create database → Standard create.
   - Engine options: MySQL → Edition: MySQL community → Version: 8.0.32.
   - Templates: Free tier.
   - Settings: DB instance identifier (e.g. studentdb-instance), Master username, Password.
   - Storage and Connectivity: leave default.
   - Public access: Yes.
   - Availability zone: No preference.
   - Click Create database.

3. Get Database Details
   - Click on DB instance identifier → Note Endpoint and Port.

4. Connect using MySQL Workbench
   - Open MySQL Workbench → Click "+" to create new connection.
   - Connection Name: any name (e.g. RDS StudentDB).
   - Hostname: Paste Endpoint.
   - Port: 3306 (default).
   - Username: Master username.
   - Store password → Enter password.
   - Test Connection.

5. If connection error:
   - AWS Console → RDS → Your DB → Connectivity & Security → VPC Security groups → Click group link.
   - Security Groups page → Inbound rules → Edit inbound rules.
   - Add rule: Type MySQL/Aurora, Source: My IP.
   - Save rules.
   - Retry Test Connection in MySQL Workbench.

6. Use database and SQL queries
   - Open connection in MySQL Workbench → Open SQL command line.
   - Perform CRUD operations on Student Database.
